Piping Tools Webservice v.1.0

Usage:

  • Start using the webservice: http://www.catalogueoflife.org/piping/webservice
  • Get all species names and annotations for a GBP: http://www.catalogueoflife.org/piping/webservice/gbp/[gbp name]/all e.g. http://www.catalogueoflife.org/piping/webservice/gbp/GBIF/all
  • Get annotated species names only for a GBP: http://www.catalogueoflife.org/piping/webservice/gbp/[gbp name]/annotated e.g. http://www.catalogueoflife.org/piping/webservice/gbp/GBIF/annotated
  • Get species and names piped to a GSD: http://www.catalogueoflife.org/piping/webservice/gsd/[gsd name] e.g. http://www.catalogueoflife.org/piping/webservice/gsd/FishBase
  • Get piping statistics: http://www.catalogueoflife.org/piping/webservice/stats
