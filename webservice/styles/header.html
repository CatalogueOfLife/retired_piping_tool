Piping Tools Webservice v.0.9

Usage:

  • Start using the webservice: /webservice
  • Get all species names and annotations for a GBP: ../webservice/gbp/[gbp name] e.g. /webservice/gbp/GBIF
  • Get species and names piped to a GSD: ../webservice/gsd/[gsd name] e.g. /webservice/gsd/FishBase
  • Get or subscribe to piping statistics and notifications feed: /webservice/rss
